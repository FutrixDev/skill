# FutrixDev Skills

This repository collects reusable Codex skills maintained by FutrixDev.

## Skills

- `wechat/` - WeChat public account publishing and article formatting skill.
- `manga-director/` - Full-color Japanese manga production skill for story planning, character references, storyboard design, prompt preview, GPT Image2 page generation, continuity checks, and revision workflows.
